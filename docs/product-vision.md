# Product Vision: Simple Terminal Tic Tac Toe

## Goal
Create the simplest possible Tic Tac Toe game that runs entirely in the terminal.

## Core Features (MVP)
- 3x3 grid displayed in text
- Two players taking turns (X and O)
- Players enter positions using numbers 1-9
- Automatic win detection (rows, columns, diagonals)
- Draw detection when board is full
- Clear board reprint after every move

## What We Are NOT Building
- AI / computer opponent
- Graphical interface
- Score tracking or history
- "Play again" prompt
- Fancy animations or colors

## Success Criteria
- Game works with one command: `python tic_tac_toe.py`
- Code stays minimal and easy to understand
- Anyone can play it immediately in any terminal
- No external dependencies
